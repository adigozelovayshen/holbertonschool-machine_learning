# Advanced Linear Algebra

This project covers advanced concepts in linear algebra, including:
- Determinants
- Cofactors
- Minors
- Adjugate matrices
- Inverse matrices
- Eigenvalues and Eigenvectors

## Learning Objectives
At the end of this project, I am expected to be able to explain:
- What is a determinant and how to calculate it
- What is a minor, a cofactor, and an adjugate matrix
- How to calculate the inverse of a matrix
- What is a characteristic equation
- What are eigenvalues and eigenvectors
