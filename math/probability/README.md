# Probability - Poisson Distribution

This project involves the study and implementation of probability distributions using Python. 

## Learning Objectives
* What is a probability distribution?
* What is a Poisson distribution?
* How to estimate the parameters of a Poisson distribution from data.
* How to calculate the PMF and CDF for a Poisson distribution.

## Requirements
* All files are interpreted on Ubuntu 16.04 LTS using `python3` (version 3.5).
* All files must conform to `pycodestyle` (version 2.5).
* All modules, classes, and functions must have documentation.
